# COVID-19 Time Series Analysis Project

## Objective
To analyze and forecast COVID-19 confirmed cases and deaths in the US using time series analysis.

## Dataset
Johns Hopkins COVID-19 US county-level dataset (confirmed and deaths).

## Methods Used
- Data visualization
- Rolling mean and standard deviation
- Dickey-Fuller stationarity test
- ARIMA forecasting model

## Tools
Python, Pandas, Matplotlib, Statsmodels

## Output
- Trends of confirmed cases and deaths
- Forecast of future cases using ARIMA
- Insights on growth patterns across time
